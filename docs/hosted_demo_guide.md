# Hosted Demo Guide

Use this checklist after deploying the sandbox.

- [ ] Bundled sample loads.
- [ ] Upload sample works.
- [ ] Top K greater than loaded candidate count does not crash.
- [ ] Ranking demo runs.
- [ ] Validation tab works.
- [ ] CSV download works.
- [ ] XLSX download works.
- [ ] Full dataset warning is visible.
- [ ] App does not require API keys.
- [ ] Mobile layout is acceptable.
- [ ] Desktop layout is acceptable.

The sandbox should remain a small-sample sanity check. Full-data ranking stays on the command line with `rank.py`.
